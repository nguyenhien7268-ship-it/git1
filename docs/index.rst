Overview
--------

The Lottery Analysis System is a Python-based application that provides:

* **Data Management**: Parse and store lottery historical data
* **Machine Learning**: AI-powered predictions using XGBoost
* **Backtesting**: Test strategies with historical data
* **Bridge Analysis**: Advanced pattern analysis for N1 and K2N modes
* **Dashboard**: Real-time analytics and predictions
* **Multi-threading**: Non-blocking UI operations
