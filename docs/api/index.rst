Module Overview
---------------

The system is organized into the following major components:

Backtester
~~~~~~~~~~

Functions for running N1 and K2N backtests with historical data.

See :doc:`backtester` for details.

Machine Learning
~~~~~~~~~~~~~~~~

AI-powered prediction using XGBoost and Q-Features.

See :doc:`ml_model` for details.

Dashboard
~~~~~~~~~

Real-time analytics, predictions, and scoring functions.

See :doc:`dashboard` for details.

Bridges
~~~~~~~

Pattern analysis functions for classic, memory, and custom bridges.

See :doc:`bridges` for details.

Data Management
~~~~~~~~~~~~~~~

Database access, data parsing, and repository functions.

See :doc:`data_management` for details.

Configuration
~~~~~~~~~~~~~

Configuration management and constants.

See :doc:`configuration` for details.
