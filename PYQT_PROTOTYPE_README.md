# PyQt6 Prototype - Quick Comparison Guide

## Running Both Versions

### Tkinter (Current)
```bash
python main_app.py
```

### PyQt6 (New Prototype)
```bash
python main_app_pyqt.py
```

## Visual Improvements in PyQt6

### 1. **Better Font Rendering**
- Smoother anti-aliasing
- Better DPI scaling on high-res displays
- Native system font integration

### 2. **Modern Styling**
- Clean, iOS-inspired design  
- Smooth hover effects
- Better button states (pressed, hover, disabled)

### 3. **Performance**
- Faster rendering
- Smoother animations
- Better memory management

### 4. **Responsive Layout**
- Better resizing behavior
- Flexible spacing
- Modern group boxes with titles
